# Authentication and Authorization

## Requirements

- Create a mock Super Admin user in `~/constants/user.ts`.
- Create two Super Admin users:
  - **Hotel Super Admin**
  - **Factory Super Admin**
- Separate the dashboard layouts:
  - **Hotel Dashboard Layout**
  - **Factory Dashboard Layout**
- When a user logs in as **Hotel Super Admin**, redirect them to the **Hotel Dashboard Layout**.
- When a user logs in as **Factory Super Admin**, redirect them to the **Factory Dashboard Layout**.
